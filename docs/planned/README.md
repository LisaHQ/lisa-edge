# Planned Capabilities

This section records ideas that are not deployable today. Planned pages describe
intent, not operator instructions, and their names are not valid service keys.

Current service plans:

- [DNS helpers](services/dns.md)
- [NUT / UPS](services/nut.md)
- [Reverse proxy](services/reverse-proxy.md)

A capability moves to `docs/services/` only after it has a canonical owner under
`services/`, a registry entry, provisioning or configuration, health coverage
and backup/recovery guidance where state is persistent.
